# @vividcodeai/embedded-xlink-darwin-arm64

Prebuilt binary for macOS Apple Silicon. This is a platform dependency of `@vividcodeai/embedded-xlink-mcp`.

**Do not install this package directly.** Install the parent package instead:

```bash
npm install @vividcodeai/embedded-xlink-mcp
```
